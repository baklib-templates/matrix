# Directify 模板使用说明文档

## 1. 模板概览 (Theme Overview)

### 模板名称

**目录站、数据集、文档库**

### 模板描述

Directify 是一个专为构建同类型多分类的数据集库、目录、各种大型集合类知识库而设计的 Baklib 主题模板。该模板提供了完整的文档管理、分类导航、搜索检索等功能，支持多产品文档库的集中管理和展示。

模板采用现代化的响应式设计，支持亮色和暗色主题切换，提供两种预设配色方案，能够满足不同品牌和场景的视觉需求。通过灵活的页面模板系统，用户可以轻松构建产品文档库、帮助中心、知识库、数据集目录等多种类型的站点。

### 适用场景

Directify 模板特别适合以下业务场景：

1. **多产品文档库**
   - 适用于需要管理多个产品文档的企业和组织
   - 支持产品分类、版本管理、独立文档展示
   - 提供清晰的产品导航和文档组织结构

2. **数据集目录站**
   - 适合构建各类数据集、资源库的目录站点
   - 支持多级分类和标签系统
   - 便于用户快速浏览和检索数据资源

3. **大型集合类知识库**
   - 适用于企业知识库、帮助中心、FAQ 站点
   - 支持丰富的文档类型（富文本、HTML）
   - 提供强大的搜索和导航功能

4. **产品手册和帮助文档**
   - 适合制作产品使用手册、技术文档
   - 支持版本管理和多语言切换
   - 提供清晰的文档阅读体验

5. **目录索引站点**
   - 适合构建各类资源、链接、内容的目录索引
   - 支持分类展示和标签筛选
   - 便于用户快速定位所需内容

### 主要特性

Directify 模板的核心功能特性包括：

#### 1. 多级分类导航系统
- 支持无限层级的分类结构
- 提供折叠式导航菜单，优化空间利用
- 左侧固定导航栏，支持粘性定位（Sticky）
- 清晰的分类层级展示，便于用户快速定位

#### 2. 强大的搜索功能
- 全站搜索支持，可检索所有文档内容
- 热门搜索关键词配置，提升用户体验
- 搜索框智能展开/收缩动画效果
- 支持搜索参数保留和筛选

#### 3. AI 智能助理
- 集成 AI 智能搜索助理功能
- 支持多轮对话式搜索
- 实时回答用户问题
- 基于文档内容的智能推荐

#### 4. 灵活的页面模板系统
- **首页模板（index）**: 展示产品文档库概览，支持 Hero 区域自定义
- **栏目模板（channel）**: 用于分类列表展示，支持多级分类
- **页面模板（page）**: 产品/公司/项目详情页面，支持富文本和 HTML 内容
- **详情模板（post）**: 通用详情页面，支持外部链接跳转
- **标签模板（tag）**: 标签分类页面
- **搜索模板（search）**: 搜索结果展示页面

#### 5. 响应式设计
- 完美适配桌面端、平板和移动设备
- 采用 Tailwind CSS 框架，确保跨设备一致性
- 灵活的网格布局系统
- 移动端优化的导航和交互

#### 6. 主题定制能力
- 支持亮色和暗色主题切换
- 提供 2 种预设配色方案
- 可自定义 Logo（亮色/暗色版本）
- 支持顶部导航栏和底部区域颜色自定义
- 丰富的 CSS 变量系统，便于深度定制

#### 7. 多语言支持
- 内置中文简体（zh-CN）和英文（en）语言包
- 支持语言切换功能
- 所有配置项均提供多语言翻译

#### 8. 内容管理功能
- 支持富文本编辑器和 HTML 代码两种内容输入方式
- 支持图片、图标、Logo 上传和外部链接
- 支持标签系统和关键词配置
- 支持文档描述和元数据管理

#### 9. 分页和列表展示
- 支持文档列表分页展示
- 每页可配置显示数量（默认 32 条）
- 美观的分页控件
- 网格布局和列表布局切换

#### 10. 面包屑导航
- 自动生成面包屑导航路径
- 帮助用户了解当前位置
- 支持快速返回上级页面

#### 11. 外部链接支持
- 支持配置外部网址跳转
- 可设置新窗口打开外部链接
- 适用于整合外部资源

#### 12. 站点导航和底部信息
- 可配置全站导航链接
- 支持底部描述、版权信息、备案号
- 可自定义底部导航栏链接

#### 13. 反馈功能
- 集成反馈按钮和反馈表单
- 支持用户提交问题和建议
- Turbo Stream 实时更新

#### 14. 其他实用功能
- 代码复制到剪贴板
- 图片查看器
- 目录（TOC）自动生成
- 滚动定位功能
- 加载更多内容
- 视频播放器支持

## 2. 模板配置说明

### 全局配置

模板的全局配置位于 `config/settings_schema.json`，主要包括：

#### 品牌相关设置
- **亮色 Logo 图片**: 建议尺寸 300x80
- **暗色 Logo 图片**: 建议尺寸 300x80

#### 联系方式设置
- **联系地址/电话**: 多行文本，可填写联系地址和电话等信息
- **全站导航链接**: 显示在左侧的全站导航链接

#### 顶部导航栏设置
- **顶部导航栏背景色**: 默认 #0d9488
- **顶部导航栏文字颜色**: 默认 #ffffff
- **顶部导航栏**: Head HTML 配置，可填写导航栏链接

#### 网站底部设置
- **底部背景色**: 建议使用深色背景，如 #181a2a
- **底部文字色**: 建议使用浅色文字，如 #e5e7eb
- **网站描述**: 一句话描述公司或产品的理念
- **版权信息**: 如：某公司版权所有
- **备案号**: ICP 备案号
- **底部导航栏**: 多行文本，可填写导航栏链接

### 页面模板配置

#### 首页模板（index.liquid）
- **Hero HTML 内容**: 自定义首页首屏区域内容

#### 栏目模板（channel.liquid）
- **标题**: 栏目标题
- **描述**: 栏目描述信息
- **内容**: 栏目详细内容（富文本）

#### 页面模板（page.liquid）
- **导入批次/版本**: 版本标识
- **公司/产品名称**: 页面标题
- **品牌名称**: 品牌标识
- **外部网址**: 外部链接地址
- **描述**: 页面描述
- **标签**: 多选标签
- **关键词**: SEO 关键词
- **ICON 图标**: 1:1 比例，建议 200x200
- **ICON 图标外部链接**: 外部图标 URL
- **LOGO 图片**: 16:9 比例，建议宽度 200px
- **LOGO 图标外部链接**: 外部 Logo URL
- **HTML 页面内容**: HTML 格式内容
- **内容**: 富文本内容

#### 详情模板（post.liquid）
- **标题**: 页面标题
- **描述**: 页面描述
- **标签**: 多选标签
- **内容类型**: 选择富文本或 HTML
- **HTML 页面内容**: HTML 格式内容（当内容类型为 HTML 时）
- **内容**: 富文本内容（当内容类型为富文本时）

## 3. 技术架构

### 前端技术栈
- **CSS 框架**: Tailwind CSS
- **UI 组件库**: DaisyUI
- **JavaScript 框架**: Stimulus（通过 Rails 集成）
- **图标库**: RemixIcon
- **动画库**: Alpine.js（用于交互效果）

### 模板引擎
- **Liquid**: Baklib 使用的模板语言
- 支持查询、分页、条件渲染等高级功能

### 文件结构
```
directify/
├── assets/              # 静态资源
│   ├── fonts/          # 字体文件
│   ├── images/         # 图片资源
│   ├── javascripts/    # JavaScript 文件
│   └── stylesheets/    # 样式文件
├── config/             # 配置文件
│   └── settings_schema.json  # 全局配置
├── layout/             # 布局文件
│   ├── theme.liquid    # 主题主布局
│   └── simple.liquid   # 简单布局
├── locales/            # 多语言文件
│   ├── zh-CN.json      # 中文翻译
│   ├── zh-CN.schema.json  # 中文配置翻译
│   ├── en.json         # 英文翻译
│   └── en.schema.json  # 英文配置翻译
├── presets/            # 预设内容
├── snippets/           # 可复用代码片段
├── src/                # 源代码
│   ├── javascripts/    # JavaScript 源码
│   └── stylesheets/    # 样式源码
├── statics/            # 静态页面
├── templates/          # 页面模板
└── docs/               # 文档目录
```

## 4. 使用建议

### 内容组织建议

1. **分类结构规划**
   - 建议采用 2-3 级分类结构，避免层级过深
   - 每个分类设置清晰的描述信息
   - 合理使用标签进行内容标记

2. **文档命名规范**
   - 使用清晰、描述性的标题
   - 为每个文档添加适当的描述和关键词
   - 合理使用标签进行分类

3. **图片资源优化**
   - Logo 建议使用 SVG 格式，保证清晰度
   - 背景图片建议使用 WebP 格式，优化加载速度
   - 图标建议尺寸统一，保持视觉一致性

### 性能优化建议

1. **图片优化**
   - 使用适当的图片格式（WebP、SVG）
   - 控制图片文件大小
   - 使用懒加载技术

2. **内容分页**
   - 合理设置每页显示数量（建议 20-32 条）
   - 避免单页加载过多内容

3. **搜索优化**
   - 合理设置热门搜索关键词
   - 为文档添加适当的关键词和标签

### 品牌定制建议

1. **配色方案**
   - 根据品牌色选择或自定义配色方案
   - 确保文字和背景有足够的对比度
   - 测试亮色和暗色模式下的显示效果

2. **Logo 设计**
   - 准备亮色和暗色两个版本的 Logo
   - 确保 Logo 在不同背景下都清晰可见
   - 建议使用透明背景的 PNG 或 SVG 格式

3. **导航栏配置**
   - 合理规划顶部导航栏链接
   - 保持导航栏简洁，避免过多链接
   - 使用清晰的导航文案

## 5. 常见问题

### Q: 如何切换主题配色方案？
A: 在 Baklib 后台的主题设置中，可以选择预设的配色方案，或通过 CSS 变量自定义颜色。

### Q: 如何添加新的页面模板？
A: 在 `templates/` 目录下创建新的 `.liquid` 文件，并在文件底部添加 `{% schema %}` 配置区域。

### Q: 如何自定义搜索功能？
A: 搜索功能已集成在模板中，可以通过配置热门搜索关键词来优化搜索体验。AI 智能助理功能需要在 Baklib 后台启用。

### Q: 如何配置多语言？
A: 模板已内置中英文语言包，可以在 Baklib 后台的语言设置中切换。如需添加其他语言，需要在 `locales/` 目录下添加对应的语言文件。

### Q: 如何自定义样式？
A: 可以通过修改 `src/stylesheets/` 目录下的样式文件来自定义样式。建议使用 CSS 变量系统进行颜色定制。
